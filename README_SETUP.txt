========================================
 DM чекер — Инструкция
========================================

ЧТО ДЕЛАЕТ СКРИПТ:
- Берёт список ников из users.txt
- Заходит на каждый профиль через AdsPower
- Если DM открыты — сохраняет ник в open_dm.txt (для ручной отправки сообщений)
- Если DM закрыты — подписывается (макс 12 на профиль) + лайкает посты 1, 3, 5, 7
- Статус и прогресс летит в Telegram

========================================
УСТАНОВКА (один раз):
========================================

1. Установи Node.js (https://nodejs.org) — версия 18+
2. Открой терминал, перейди в папку скрипта:
   cd путь/к/папке
3. Установи зависимости:
   npm install
4. Готово

========================================
НАСТРОЙКА (перед первым запуском):
========================================

Открой index.js и заполни:

1. PROFILES (строки 7-12) — впиши 4 ID профиля из AdsPower:
   const PROFILES = [
     'твой_id_1',
     'твой_id_2',
     'твой_id_3',
     'твой_id_4',
   ];

2. API_KEY (строка 15) — API ключ из AdsPower (уже заполнен, замени если другой)

3. TG_TOKEN и TG_CHATS (строки 17-18) — токен бота и ID чатов Telegram

4. Положи ники в users.txt — по одному на строку, с @ или без:
   elonmusk
   @durov
   jack

========================================
ЗАПУСК:
========================================

   node index.js

========================================
РЕЗУЛЬТАТ:
========================================

- open_dm.txt  — ники с открытыми DM (пиши им руками)
- done.txt     — все обработанные ники
- failed.txt   — ники с ошибками (при повторном запуске попробует снова)
- log_*.txt    — подробные логи

========================================
ПОЛЕЗНОЕ:
========================================

- Ctrl+C — корректная остановка (дожидает текущего юзера)
- Повторный запуск — продолжит с того места где остановился
- Лимит подписок: 12 на каждый профиль (настройка MAX_FOLLOWS_PER_PROFILE в index.js)
- AdsPower должен быть запущен и профили авторизованы в X

========================================
ТРЕБОВАНИЯ:
========================================

- Node.js 18+
- AdsPower запущен на localhost:50325
- Профили авторизованы в X.com
- Интернет
